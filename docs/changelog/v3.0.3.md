# Version 3.0.3

- Added Wordpress preset for breakpoints, which provides the following settings: 

```js
app.use(VitalMq, {
    preset: 'wordpress'
})
```

| Name    | Minimum width |
| ------- | ------------- |
| mobile  | 0            |
| small   | 600          |
| medium  | 782          |
| large   | 960          |
| xlarge  | 1080         |
| wide    | 1280         |
| huge    | 1440         |
